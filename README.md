# Pricing Table

A **responsive pricing table** built using **HTML and CSS**, designed to showcase different subscription or product plans. This project demonstrates layout structuring, responsive design, and attention to detail in frontend web development.

## Features
- Clean, modern design with clear plan distinction
- Responsive layout compatible with desktop, tablet, and mobile screens
- Highlighted features for each pricing plan
- Call-to-action buttons for purchase or signup
- Styled using **CSS Flexbox and Grid** for flexibility

## Technologies Used
- HTML5
- CSS3
- Flexbox & CSS Grid
- Responsive Design
